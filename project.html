[ 01 ] — the brief

Make a kit actually drive itself.

Self-driving feels like magic — so I started with a robotics kit as the rolling chassis and focused where the real learning is: the sensors and the code that turn raw LiDAR into decisions.

The current build is a compact rover with a spinning LiDAR for 360° range data, a microcontroller brain, and a battery pack — the platform I'm using to experiment with mapping, obstacle avoidance, and eventually full autonomy.

Autonomy is still very much a work in progress — this is the rig I learn on.

Build Process

CAD → PRINT → MOVE
STEP 01

Start from the kit

Assembled the kit chassis, motors, and wheels into a solid rolling base — the platform I build the sensing and autonomy on top of.

STEP 02

Wire the brains

Mounted the battery, motor drivers, and controller, then ran the harness and got the LiDAR spinning and streaming range data over the bus.

STEP 03

Teach it to see

Now the hard part: turning LiDAR scans into a map, avoiding obstacles, and inching toward real autonomy. This is the part that's still very much in progress.
